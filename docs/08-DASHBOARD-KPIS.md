## Bubble Implementation Notes

### Repeating Groups
- Lead Pipeline: Repeating group of Leads, sorted by score descending
- Activity Timeline: Repeating group of Activities filtered by Lead, sorted by date descending
- Daily Trend: Use Bubble chart plugin with DailyMetric data

### Conditional Formatting
- Score badges: Red (<0), Gray (0-20), Yellow (21-40), Orange (41-60), Green (61+)
- Stage pills: Color-coded option set values
- Alert indicators: Pulsing for hot/qualified leads requiring action

### Real-Time Updates
- Use Bubble's "Do every 30 seconds" on dashboard to refresh KPI numbers
- Activity feed on lead detail page auto-refreshes
- Notification bell with unread count in header
